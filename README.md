## Getting Started

Want to run this locally? Here's how:

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the magic happen.

```bash
# Build for production (if you're into that)
npm run build
npm start
```
